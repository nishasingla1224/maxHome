# Email Inbox (React + TypeScript)

Simplified Email Inbox UI with partner-level feature toggles.

## Run locally
```bash
npm install
npm run dev
```
Then open http://localhost:5173

## Features
- Inbox list with sender, subject, date, snippet
- Bulk actions (Mark Read/Unread, Spam, Delete)
- Email detail view with reply UI
- Partner configs (A, B) toggle features and theme
